# RainWeather

#### 介绍
1.本软件为安卓平台软件
2.通过高德地图定位获取当前的地理信息，通过地理信息获取天气信息和PM2.5信息
3.主要功能，当日天气，近期天气预报，当日空气质量，24小时天气预报
